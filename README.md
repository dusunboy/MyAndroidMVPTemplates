# MyAndroidMVPTemplates

我的AndroidDagger2MVP框架模板和项目框架生成工具

## 使用
使用Android Studio运行项目,在MyClass类中使用右键找到MyClass.main()运行<br>
生成的模板保存在build下
有两个模板生成

## 1. 项目框架生成
项目框架模板放在resources/MyProjectFramework

## 2. Dagger2 MVP模块生成
Dagger2 MVP模板放在resources/dagger2
